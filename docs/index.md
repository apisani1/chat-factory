# chat-factory

A short description of the project

## Overview

This documentation covers the chat-factory library.

## Installation

```bash
pip install chat_factory
```

Or, if you use Poetry:

```bash
poetry add chat_factory
```

## Quick Start

```python
from chat_factory import Example

# Initialize
example = Example()

# Use the library
result = example.run()
print(result)
```
